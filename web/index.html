Welcome to the PidFile project page

PidFile allows you to make sure that there is only one instance of a program running at any point in time. This is achieved with a pid file that records the pid of a program. When it starts any existing pid file is checked to see if the program that created it is still running. If so the calling program can halt or take remedial action.

One way to use it

require 'pidfile'

unless Pid::PidFile.set_pid("/var/run/app.pid")
  puts "This process is already running!"
  exit
end

# ...the rest of your program goes here

Or in a more Ruby like way

require 'pidfile'

Pid::PidFile.new("/var/run/app.pid") do
  # ...the rest of your program goes here
end

If an instance is already running then it will throw a Pid::AlreadyRunning exception
